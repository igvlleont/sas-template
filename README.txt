sas-conf.yml - конфигурация автономного сервера

create-db-from-cf.bat - создать базу данных на основе файла выгрузки конфигурации CF

create-db-from-dt.bat - создать базу данных на основе файла выгрузки базы данных DT

load-update-cf.bat - обновить базу из CF файла, обновить конфигурацию базы данных

load-update-dt.bat - обновить базу из DT файла

server-start.bat - запуск сервера

designer-start.bat - запуск конфигуратора


Использование: 

1) выделить папку для экземпляра автономного сервера и скопировать туда все файлы скриптов и файл конфигурации сервера

2) заполнить настройки в sas-conf.yml

3) Воспользоваться скриптом create-db-from-cf.bat для создания базы из cf или create-db-from-dt.bat для создания базы из dt. 
   После запуска скрипта необходимо в открывшемся окне консоли указать расположение файла cf или dt

4) Запустить сервер через server-start.bat

5) Запустить конфигуратор базы через designer-start.bat
